# Chronic_Kidney_Disease_Prediction_using_Machine_Learning

📌 Project Overview
This project aims to predict Chronic Kidney Disease (CKD) using Machine Learning techniques. 
Early detection of CKD can help in timely medical intervention and better patient care. 
The model analyzes various medical attributes to classify whether a patient has CKD or not.

📊 Dataset Information
Source: UCI Machine Learning Repository
Total Records: 400
Features: 24 medical attributes
Target Variable: CKD / Not CKD
The dataset includes important clinical features such as hemoglobin, serum creatinine, albumin, packed cell volume, blood pressure, and hypertension.
